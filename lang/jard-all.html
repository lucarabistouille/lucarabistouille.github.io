  • gelb

  • die Glocke, la cloche

    • Mot d’origine peu claire, qui aurait été répandu par les moines irlandais, qui portaient des cloches. Cf. français cloche, latin médiéval clocca, anglais ancien clucge qui a donné clock
    • diminutif, das Glöckchen
    • dans le contexte liturgique, pl. die Glocken, ou die Altarglocke pour les clochettes, et le coup de cloche se dit das Glockenzeichen, traduction proche du latin signum dare
    • Quelques dérivés usuels :
      • das Glockenspiel, le carillon ou le glockenspiel
      • die Glocken lauten, les cloches sonnent ; Es hat zur Pause geläutet, Ça a sonné pour la pause (à l’école)
        • le carillon se dit ainsi aussi das Glockengelaüt
      • die Kirchenglocke, la cloche (d’une église) et der Glockenturm, le clocher
        • d’où l’expression auf den/mit dem Glockenschlag, à l’heure sonnante (der Glockenschlag, le coup de cloche), où, dans ce contexte anschlagen, frapper (une cloche)
